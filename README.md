# Smart Calculator Pro

Smart Calculator Pro is a modern and responsive calculator web application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations along with additional features such as scientific calculations, keyboard support, calculation history, theme switching, and error handling.

## Features

- Basic arithmetic operations (+, −, ×, ÷)
- Percentage calculations
- Square (x²) and Square Root (√) functions
- Keyboard input support
- Calculation history tracking
- Dark/Light mode toggle
- Error handling for invalid expressions
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Files

- `index.html` – Structure of the application
- `style.css` – Styling and responsive design
- `script.js` – Calculator logic and functionality

## How to Use

1. Open the application in a web browser.
2. Enter numbers and operators using the buttons or keyboard.
3. Press `=` or `Enter` to calculate.
4. Use `AC` to clear the display.
5. Use `⌫` to delete the last character.
6. Switch between Dark and Light mode using the theme button.
